== Installtion ==

1.)Put directory inside wp-content/plugins folder
2.)Activate plugin from admin panel
3.)You will see a tab "Auth Client" in dashbord.
4.)Click "Auth Client" menu from dashbord it open a page to save auth key.
5.)Paste the "auth key(generated from remote server)" here.
6.)Click on Save Changes


== Note ==

This plugin will not work individual you need to active (Auth Server) plugin at remote site.




== What Basically does both plugin  ==

On the basis of both the plugin we are copy all user from one website(Remote Server) to other website(Client Server) so that if user register on remote site then
user no need to create another account on client server.

User can automatically login on client server using the same crediential of remoter server.

== Installtion ==

1.)Put directory inside wp-content/plugins folder
2.)Activate plugin from admin panel
3.)You will see a tab "Unique Token" in dashbord.
4.)Click "Unique Token" tab from dashbord it open a page to save token.
5.)Paste the "generated token from remote server for this domain" here.
6.)Click on Save


== Note ==

This plugin will not work individual you need to active (Generate Token) plugin at remote site.
Bcoz the generated token key we will use for this plugin to send userinfo from remote server to client server.



== What Basically does both plugin  ==

On the basis of both the plugin we are copy all user from one website(Remote Server) to other website(client server) so that if user register on remote site then
user no need to create another account on client server.

User can automatically login on clinet server using the same crediential of remoter server.
